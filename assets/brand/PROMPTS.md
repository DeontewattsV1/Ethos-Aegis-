# Ethos Aegis Prompt Pack

## Master generation prompt

Create a government-grade black-ops brand identity system for “Ethos Aegis,” a sovereign AI integrity defense architecture. The visual identity must feel like a high-trust national security platform built to confront an AI apocalypse, but it must be completely original and must not copy any real military seal, insignia, or protected government mark. Design it as a dark, authoritative, elite system with a black-on-black foundation, deep charcoal textures, steel-blue highlights, and precise white typography. The mood should be vigilant, doctrinal, and unbreakable, like an emergency defense command created to sanitize corrupted intelligence and restore order.

Develop multiple singular brand assets that work independently and together as one cohesive system: a primary emblem, a compact icon, a wordmark, a seal-style lockup, a mission patch variation, a horizontal header mark, a social avatar version, a favicon-style symbol, and a field-ready badge. The primary symbol should combine abstract shield geometry, a protective mesh, a halo ring, or a fortified aperture motif, with severe symmetry and subtle tactical precision. It should suggest moral judgment, containment, resilience, and purification without using literal weapons, eagles, stars, flags, or any existing defense insignia. The icon language should feel like an adaptive immune architecture translated into a sovereign defense mark.

Typography should be condensed, disciplined, and modern, with a strong institutional presence. The wordmark should be highly legible at small sizes and feel like it belongs on a secure operations platform, a classified dossier, or a strategic defense dashboard. Use a palette dominated by matte black, graphite, obsidian, smoke gray, and one restrained accent such as cold steel blue or signal red used sparingly for alert and authority. Lighting should be dramatic, low-key, and cinematic, with controlled highlights that reveal texture and depth. Surfaces should feel like anodized metal, embossed rubber, laser-etched polymer, and precision-printed card stock.

Design the brand system to look premium, severe, and instantly recognizable from a distance. It should command attention on GitHub, in documentation, on startup decks, in app icons, on seals, and on banners. Create variants that include: a minimalist icon, a full emblem, a stacked wordmark, a circular badge, a monochrome utility version, and a reverse-white version for dark backgrounds. Everything should feel like it belongs to a hardened AI defense initiative that exists to end chaos, expose corruption, and protect the integrity of machine judgment. The final result should be cinematic, iconic, and unmistakably sovereign.

## Tight version

Design a sovereign black-ops brand identity for “Ethos Aegis,” an elite AI defense architecture. Make it feel like a classified government response system to an AI apocalypse: severe, precise, unyielding, and instantly credible. Use black-on-black surfaces, graphite, steel-blue highlights, sharp symmetry, and modern condensed typography. Create multiple standalone assets: primary emblem, icon, wordmark, seal-like badge, and app mark. The symbol must be original and abstract, inspired by protection, containment, judgment, and purification, not any real military insignia. Make it look premium, cinematic, and operational, with a visual language that commands attention in a GitHub repo, security console, or defense brief.

## Negative prompt / guardrails

Do not use or imitate any real government seal, military branch insignia, Department of Defense mark, flags, eagles, stars, rank marks, literal weapons, official department names, country names, national coats of arms, or protected badges. Avoid designs that could be mistaken for official state authority. Keep the mark abstract, civilian, original, and brand-owned.

## Individual asset prompts

### Primary emblem
Create a primary Ethos Aegis emblem using an abstract shield, fortified aperture, immune mesh, and halo-ring geometry. Severe symmetry. Black-on-black, steel-blue highlight, white typography. Original, not a real seal.

### Monogram icon
Create a compact EA monogram for Ethos Aegis. Make it legible at favicon size. Use an abstract containment ring and shield aperture. No flags, eagles, stars, or weapons.

### Wordmark lockup
Create a horizontal Ethos Aegis wordmark lockup for documentation and GitHub headers. Condensed institutional typography, steel-blue accent, severe spacing, premium black field.

### Circular seal-style badge
Create a circular badge-inspired lockup for Ethos Aegis using original abstract geometry only. It should feel formal and high-trust without copying military or government seals.

### Mission patch
Create a hexagonal or angular mission patch for Ethos Aegis. Use original containment geometry, no military units, no national symbols, no weapon motifs.

### App icon
Create a square rounded app icon for Ethos Aegis. Minimal fortified aperture on matte black. Must read clearly at small size.

### Reverse mark
Create a monochrome white Ethos Aegis utility mark for dark backgrounds. Simple, clean, precise, and scalable.

### Documentation header banner
Create a cinematic GitHub/documentation header banner for Ethos Aegis with the emblem, wordmark, tagline, and subtle tactical grid texture.
